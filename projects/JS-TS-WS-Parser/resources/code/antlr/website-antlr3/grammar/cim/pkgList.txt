The cim22Grammar package list
-----------------------------

The cim22Grammar package v0.2, December, 2001, includes the following
files:

1) An ANTLR 2.7.1 CIM Specification v2.2 grammar file - cim22Grammar.g
2) A Java main file - cim22.java
3) Two MS-DOS batch files - run.bat and go.bat
4) A mini-howto - cim22Grammar.pdf
5) A copy of the CIM Schema v2.6 directory - 2.6\
