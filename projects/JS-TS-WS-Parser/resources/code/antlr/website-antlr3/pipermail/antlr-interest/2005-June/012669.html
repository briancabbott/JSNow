A shot in the dark: remove the third alternative and replace the "+"
with "*" in the second one.  This results in:

NAME:
     ( CAPWORD (WS CAPWORD)* COMMA WS CAPWORD )
            => CAPWORD (WS CAPWORD)* COMMA WS CAPWORD
     | ( CAPWORD (WS CAPWORD)* )
            => CAPWORD (WS CAPWORD)*
;

Or perhaps, simply :-)

NAME:
    CAPWORD ( WS CAPWORD )* ( COMMA WS CAPWORD )?
;

You might need to make the loops greedy, but this should work.
