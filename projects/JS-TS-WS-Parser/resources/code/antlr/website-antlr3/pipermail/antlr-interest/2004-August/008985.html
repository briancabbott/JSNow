I have the following rules in my lexer:

protected
DOT     :   '.'        ;

protected
DIGIT   :   '0'..'9'   ;

NUMBER  : (DOT (DIGIT)+)
        | (DIGIT)+ (DOT (DIGIT)*)?;

of NUMBER and an interval 5..9 (NUMBER INTERVAL_SIGN NUMBER). Do I
