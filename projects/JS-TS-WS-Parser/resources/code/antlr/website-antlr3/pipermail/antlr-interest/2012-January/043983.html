elem_array: elem* EOF;

elem: SPACE // The final frontier
     (
       | LETTER_A LETTER_A
       | LETTER_B LETTER_B
       | LETTER_C LETTER_C
       | LETTER_D LETTER_D
     )
;

Should do it. You are basically trying to avoid ? in two rules, one higher
than the other, which leads to ANTLR not able to decide which path to take
when nothing is there.
