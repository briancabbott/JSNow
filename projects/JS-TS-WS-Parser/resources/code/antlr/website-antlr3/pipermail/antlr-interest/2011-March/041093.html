You need to construct the correct LL tree form of the rules. Always copy
from a pre-written grammar in the examples or read the book and you will not
struggle so much:

factor
  : addexp (('+'|'-')* addexp
  ;

addexp
   : mulexp (('*'|'/') mulexp)*
   ;

mulexp
   : '-' mulexp
   | atom
   ;

atom
   : NUMBER
   | '(' factor ')'
   ;
