tokens such as '53xx' '6xxx' '3334' and the following simple grammar

grammar.  Is there any way I can get shapeDist to work?

John

// This works
dist    
:    '^' SHAPE
;

ID : ('a'..'z'|'A'..'Z'|'_')('a'..'z'|'A'..'Z'|'_'|'0'..'9')+;
SHAPE     :(DIGIT (DIGIT|'*'|'x'|'X') (DIGIT|'*'|'x'|'X')

shapeDist
