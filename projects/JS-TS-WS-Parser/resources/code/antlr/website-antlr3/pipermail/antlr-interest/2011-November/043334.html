Ah, hold on, you're using the `..` (range) operator inside a parser rule
(horef). Either create a lexer rule matching '?'  'a'..'z':

Horef
    :    '?' 'a'..'z'
    ;

, or move  'a'..'z' to a lexer rule:

horef
    :    '?' Letter
    ;

Letter
  :  'a'..'z'
  ;


But still, the question mark should not be escaped.
