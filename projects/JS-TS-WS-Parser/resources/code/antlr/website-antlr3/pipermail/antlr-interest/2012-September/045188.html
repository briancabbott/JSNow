You need to use the parser:

line : .* NL;


L1 : 'dfddfdf';
L2 : 'dfdfdfd';
NL: '\n';
ANY : . ;


Should get you a little nearer to what you want.

> Try something like this for lines instead:
>
> SOURCE_LINE_: .* (NEWLINE_ | EOF);
>
> Then your preprocess rule could go like this:
>
> preprocess:
> 	line* EOF
> ;
>
>
