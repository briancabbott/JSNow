you can define local [non-class] variables with the @init clause.
class variables cannot be used because you will get a compile error in the
generated code.

rulename
scope GlobalOne;
@init
{
       bool localFlag;

       $GlobalOne::globalVariable = ....;
       ...
       localFlag = true;
}
