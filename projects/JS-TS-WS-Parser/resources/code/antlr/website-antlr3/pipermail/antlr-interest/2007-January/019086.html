IIRC, the greedy option needs to be in the same scope as both the 
. and the *.  Try writing it like this:

: '<verbatim>' (options {greedy=false;} :
   (a=. { verbatime.addContent(a.getText()); }
   )*) '</verbatim>'
;

(Haven't tested this, but it seems plausible.)
