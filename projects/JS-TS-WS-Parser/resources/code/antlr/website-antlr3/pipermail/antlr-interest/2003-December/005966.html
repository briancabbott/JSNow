Some possible resources:

David Wigg's upgrade of the PCCTS-based C++ parser should be usable
(<http://www.antlr.org/grammar/cpp>)

SWIG (www.swig.org) may also be usable--it includes a C++ subset 
parser.

The Ovid project (<http://ovid.tigris.org/>) has a few things that 
might help, although it is still very primitive.
