At 2004-09-29 22:31 +0000, hero_onethousand wrote:
>Is there an easy way to change the final value of a token declared
>in the tokens section of the parser definition?

Put your token enumeration in a separate ...TokenTypes.txt file and
use importVocab to reference it.

   http://www.antlr.org/doc/options.html#importVocab
