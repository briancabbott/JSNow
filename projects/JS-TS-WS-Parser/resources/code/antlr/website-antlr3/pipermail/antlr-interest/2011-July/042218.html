Hi Qiao,

With Java as the target, you could do:

SINGLE_QUOTED_STRING
@after {
  String s = getText();
  setText(s.substring(1, s.length() - 1));
}
  :  SQUOTE (~SQUOTE)* SQUOTE
  ;


Regards,

Bart.
