I assume you're trying to write a new target? If so, you can do
something like this, with nested try/catch and a duplication of the
finally code:

try {
  try {
    <tryCode>
  } catch ( ExceptionType1 ) {
    <handler>
  }
} catch ( ... ) {
  <finallyCode>
  throw;
}
<finallyCode>
