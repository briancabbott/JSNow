Things can't get any simpler than walking the tree by hand:

void walk(AST ast) {
  // do stuff
  AST child = ast.getFirstChild();
  while (child != null) {
     walk(child);
     child = child.getNextSibling();
  }
}

I'm using an old version of ANTLR. By now, it's probably:
void walk(AST ast) {
// do stuff
for (AST child: ast.getChildren()) {
   walk(child);
}
}
