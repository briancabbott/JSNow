overloaded functions?  I'm familiar with how C++ does it, but am
looking for other options.

for parameters as well as positional parameters.  Hence you might see:
   f(10, 2.0)
   f(10, b=2.0)
   f(a=10, b=2.0)
all of which are equivalent functions.
