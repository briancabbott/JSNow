Hi Andrei,
You can make ID and QW 'fragment' rules as follows to get rid of the
warning:

grammar Name;
start: wordList ;
wordList: '[' Name (',' Name)* ']';
Name:   ID | QW;
fragment ID:    ('a'..'z'|'A'..'Z')+ ;
fragment QW:    '"' ('a'..'z'|'A'..'Z') (' '|'a'..'z'|'A'..'Z')*
('a'..'z'|'A'..'Z') '"';
WS:    (' '|'\t' |'\r'|'\n')+ {skip();} ;

Regards,

Darach.
