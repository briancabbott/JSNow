Only at the end. You need to place the code-block *inside* the parenthesis
like this:

keysAndValues
  :  ('"' key '"' value
       {
         // Some code to read the key(s) and value(s) out.
         System.out.println("key=" + $key.text + ", value=" + $value.text);
       }
     )*
  ;
