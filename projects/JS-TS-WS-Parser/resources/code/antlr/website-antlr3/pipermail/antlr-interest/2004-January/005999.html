Folks,

After communicating with the inventors of the LALL class of grammars, I 
have an example grammar that is LALL but not SLL.  Furthermore, ANTLR 
does not accept it so ANTLR cannot handle the LALL class directly.  To 
set the record straight then:

PCCTS: SLL(k) + sem and syn preds with hoisting of sem preds
ANTLR 2.x: SLL_1(k) + sem and syn preds
ANTLR 3: SLL-regular(k) + sem and syn preds with hoisting of sem preds

Sorry for the confusion.
