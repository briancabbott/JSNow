Hi,

 

Here is a fix for your grammar. These things are well explained in the
ANTLR book.

 

grammar Test; 

 

site    :  page+

            ;

 

page    :            'PAGE' ID

            ;

 

ID         :           LETTER+

            ;

 

TEXT    :    LETTER ( LETTER )*

            ;

    

LETTER            :           'a'..'z'|'A'..'Z' 

            ;

 

    

WS       :           (' ' | '\t' | '\r' | '\n')+ {$channel=HIDDEN;}

            ;
