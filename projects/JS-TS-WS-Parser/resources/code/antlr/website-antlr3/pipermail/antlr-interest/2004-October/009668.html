right now I trying to attack my comment problem with a predicate:

 COMMENT
    :
        "--" 
        (
           ~('-'|'\n'|'\r'|'\u000b'|'\u000c') | {LA(2) != '-'}? '-'
        )*
        {
           // consume "end-of-comment" characters
           switch(LA(1)) {
            ..
            case '\u000b' : {        <== syntax error on '\u000b'
              match('\u000b');       <==       dito
              break;
            }
            ..
           }
        }
    ;

proposals).
