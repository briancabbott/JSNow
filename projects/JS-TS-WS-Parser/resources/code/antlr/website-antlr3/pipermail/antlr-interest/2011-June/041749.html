Lookup the latin Unicode code pages on Wikipedia and add the Unicode code
points for 
accented Latin1 to your rule WORD.

fragmen
Latin1_Supplement                   :   '\u00A0' .. '\u00FF';
fragment
Latin_ExtendedA                     :   '\u0100' .. '\u017F';
fragment
Latin_ExtendedB                     :   '\u0180' .. '\u024F';
