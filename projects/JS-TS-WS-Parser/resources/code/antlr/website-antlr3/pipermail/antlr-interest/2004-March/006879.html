I meant: use the tokens section, delete your non-ID rules and be sure 
to use testLiterals=false.

Then you can do something like this:
ID : ('a'..'z')+
{$setType(testLiteralsTable(ID));
if (_ttype==TOKEN_IFDEF).......semantic actions........ } 
;
