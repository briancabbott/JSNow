You could use predicates and match a non-identifier character after the
word (you could have a variable named trueThat or falseData).

TRUE:
	("true" ~(IDENTCHAR))=> "true" ;
FALSE:
	("false" ~(IDENTCHAR))=> "false" ;

protected IDENTCHAR:
	('a'..'z'|'A'..'Z'|'0'..'9'|'_') ;
