  Looks like you need to change the bluish part to:

             // or just a category 
             cat:( LPAREN! tCat:ID RPAREN! )
             { 
                  ## = #( #[ObjcCATEGORY, "categorie"], #tCat ); 
             }

  What you had:

                  #cat = #( #[ObjcCATEGORY, "categorie"], #tCat ); 
                  ##.addChild(#cat);

  ... first sets #cat to
      ObjCATEGORY
      + ID
  ... but ## is still ID, so "##.addChild(#cat)" then gives you:
      ID
      + ObjCATEGORY
         + ID

  Hope this helps,

  Davin
