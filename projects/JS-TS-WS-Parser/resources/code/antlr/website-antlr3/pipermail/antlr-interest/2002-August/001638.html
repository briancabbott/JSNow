Hi Developers!

Aqris Software is glad to announce that RefactorIT 1.1 is free for working
on ANTLR source code.

RefactorIT is a java refactoring tool that integrates with Netbeans, Sun ONE
Studio, Forte, JBuilder, Oracle JDeveloper and also works as standalone.
RefactorIT features include Extract Method, Move Class, Rename and Search
for every item and much more.

To check RefactorIT features please go to
http://www.refactorit.com/index.html?id=474

To see a list of Open Source projects, RefactorIT is free for, go to
http://www.refactorit.com/index.html?id=649

You can download RefactorIT from http://www.refactorit.com/ - to get a free
license that allows working on ANTLR and other selected open source projects
please register yourself as user and be sure to check 'Send me the
OpenSource Community license key' checkbox.

happy refactoring!
