It works as it should, but your expectations are not correct I am afraid. Generally you are falling over on style:


X : a* b -> ^(NODE a* b) ;

Rather than

x : a b  -> ^(NODE a* b) ;
a: x* ;

Then what you think happens does happen. It also removes extra method calls (generally).
