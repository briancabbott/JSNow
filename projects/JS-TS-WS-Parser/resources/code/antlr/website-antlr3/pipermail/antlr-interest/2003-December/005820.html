MSG_DIR and MSG_TYPE share a common prefix of '[', so you need to combine
them into one rule, something like 

MSG_DIR
 : '[' 
       ("in" | "out")
       |("asyn" | "sync") {$setType(MSG_TYPE);}
    ']'
 ;
