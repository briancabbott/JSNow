> Looks like a bug to me:
>
> grammar jsp;
>
> JAVASCRIPT    :    '<script>' ( options {greedy=false;} : . )* 
> '</script>' {System.out.print("J");};   
> ANY    :    . {System.out.print("A");};

> jsp        :    (ANY | JAVASCRIPT)* EOF;
>

>
> <script>foo</script>
> <s>bar</s>
>
>

> "<script>foo</script>", "a", "r", "<", "/", "s", ">"

>
> JAVASCRIPT, ANY, ANY, ANY, ANY, ANY, ANY
>

>

> JAAAAAAAA
