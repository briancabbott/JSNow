> Uhh, subtitles please...

> removing the brackets?

> >> // Input: 1 + 2
> >> // Generated AST: ^(+ 1 2)
> >> okExpr
> >>   :    NUM (('+'|'-')^ NUM)*



basiaclly, I think you want:

NUM ((('+')^|('-')^) NUM)*

You can also do:

NUM (arithOp^ NUM)*;

arithOp : '+' | '-' ;


Though I personally would get rid of all the literals in the parser
grammar and replace them with lexer tokens.
