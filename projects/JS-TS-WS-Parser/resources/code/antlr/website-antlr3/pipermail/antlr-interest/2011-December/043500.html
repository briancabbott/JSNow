No, that is not possible.

The closest you can come is create a fragment:

fragment HEX_DIGIT_BLOCK : HEX_DIGIT HEX_DIGIT HEX_DIGIT HEX_DIGIT;

and then do:

Unicode_escape_sequence:
 | '\\u' HEX_DIGIT_BLOCK
 ;

Bart.
