> How about passing the formal type as a parameter?
>
> formalParameters
>     :    t=TYPE formalParameterDeclsRest[$t] -> formalParameterDeclsRest
>     ;
> formalParameterDeclsRest[ Token t ]
>     :    i=IDENTIFIER
>         ( ( ',' formalParameters -> ^({$t} $i) formalParameters )
>         | ( /*empty*/            -> ^({$t} $i) )
>         )
>     ;
