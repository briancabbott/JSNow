Since you want to keep "error" out of the literals table, then treat it as
an ID:

on_error_stmt
   : "on" ID "goto" label
   ;

If you want to make sure that it is always "error" you can use a validating
semantic predicate:

on_error_stmt
   : "on" { "error".equals(LA(1).getText()) }? ID "goto" label
   ;
