Maybe you are looking for something like this:

    attributes
        :    attribute+
        ;

    attribute
        :    typeAttr
        |    lengthAttr
        |    pathAttr
        ;

Here is a complete sample:

    grammar Test;

    attributes
        :    attribute+
        ;

    attribute
        :    typeAttr
        |    lengthAttr
        |    pathAttr
        ;

    typeAttr    :    'TYPE' ID ';'
        ;
       
    lengthAttr
        :    'LENGTH' NUMBER ';'
        ;
       
    pathAttr    :    'PATH' ID ';'
        ;
       
    ID    :    'a'..'z'+
        ;
       
    NUMBER    :    '0'..'9'+
        ;
       
    WS    :    (' ' | '\t' | '\n')+ {$channel = HIDDEN;}
        ;
