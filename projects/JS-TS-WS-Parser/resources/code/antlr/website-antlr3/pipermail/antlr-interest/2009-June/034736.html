Hi Xiaohua,

Try this one:

STRING :
   '"' (~('"'|'\n'|'\r'))* '"';

It says the string starts with " and ends with " while an between  
those there can be any character but ",\n or \r
This was taken from the templates used in the ANTLRIDE editor.
