Hi,

I use the following rule for single line comments:

SL_COMMENT : "//"
             ( ~( '\n' | '\r' ) )*
             /*
              * single line comment may not
              * be followed by new line
              * at the end of file
              */
             ( NL )?
           ;

NL : (   {LA(2) == '\n'}? '\r' '\n'
       | '\r'
       | '\n'
     )
     { newline(); }
   ;
