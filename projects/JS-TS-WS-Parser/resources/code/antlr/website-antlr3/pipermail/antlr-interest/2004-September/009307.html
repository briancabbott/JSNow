There is also precompiled .net (J#) executable for antlr:
http://www.antlr.org/share/1057505418267/antlr_net-2.7.2_bin.zip
You need .net framework installed, but since you're using C# then it should
work. You just need to install J# libs (which is free distrib from
Microsoft)
