> 
> ^(TOMATCH)
> 	-> ^(RESULT1)   if TOMATCH.Text = 1
> 	-> ^(RESULT2)   if TOMATCH.Text = 2
> 	LEAVE IT UNCHANGED otherwise

> ANTLR apparently thinks something has changed and reruns the rule (and

> 		-> {$c.Text == @"\begin"}? ^(ENVIRONMENT_BEGIN $b)

> 		-> ^(COMMAND $c ^(BLOCK $b))
