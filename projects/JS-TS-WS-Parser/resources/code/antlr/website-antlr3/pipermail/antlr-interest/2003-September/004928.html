Now Pete,

You know that's nothing that ten or eleven chars
of perl can't fix.  You just need a preprocessor,
here it is:

    perl -pe 'y/{}/: /' 

as in:

$ cat >tmp
i = 2
while i {
    print "Redundancy is good."
    i -= 1
}

$ perl -pe 'y/{}/: /' tmp | python
Redundancy is good.
Redundancy is good.

To brace or indent, that is the question.
No.  You can have both.
