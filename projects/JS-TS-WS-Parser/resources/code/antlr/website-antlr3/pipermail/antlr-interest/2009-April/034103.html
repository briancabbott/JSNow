The problem is, that you use skip() where you should actually use  
something like:

: "//" (~('\r'|'\n'))* {$channel=HIDDEN;}

This way the comment will still be acessible in the parser:

http://www.antlr.org/wiki/pages/viewpage.action?pageId=557063
