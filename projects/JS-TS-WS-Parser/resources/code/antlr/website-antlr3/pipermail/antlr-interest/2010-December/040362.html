Hi Gian,

If you must use "HIDDEN" instead of "Hidden", you can place the following in
your grammar:

For a lexer grammar:

@members {const int HIDDEN = Hidden;}

For a combined grammar:

@lexer::members {const int HIDDEN = Hidden;}
