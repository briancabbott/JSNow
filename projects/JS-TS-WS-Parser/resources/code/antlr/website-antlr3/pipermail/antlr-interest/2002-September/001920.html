Hey all,

I did this a month or so ago.  The problem is that there is an assumption in
the configure script that the CLASSPATH separator was ":".  However on
Windows, it is ";" (to allow for drive letters, I guess).  If you change the
following line in the configure script, you'll be OK:

	CLASSPATH=CLASSPATH:.

to

	CLASSPATH=CLASSPATH;.

Cheers,
Doug Schaefer
