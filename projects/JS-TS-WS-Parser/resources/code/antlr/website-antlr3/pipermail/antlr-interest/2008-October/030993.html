Am 07.10.2008 16:40, Sergiy Karpenko schrieb:
> Hi, all    
> For axample we have a lexer rule:    
>     
> CHAR : ('a'..'z')|('0'..'9');    
>     
> But, I want exclude from this sequence 'k' symbol.     
> Somethig like    
> EXTCHAR: CHAR ~'k' ;    
> [...]

You can do it the other way around:

EXTCHAR: ('a'..'j')|('l'..'z')|('0'..'9');
CHAR: EXTCHAR|'k';
