Try something like this:

BLOCK_COMMENT
 : '/*' ({input.LA(1) != '*' && input.LA(2) != '/'}?=> . )*
   ( '*/'
   | EOF {System.err.println("Oi, wrong BLOCK_COMMENT!");}
   )
 ;


Note that the predicate, `{input.LA(1) != '*' && input.LA(2) != '/'}?=>`,
contains target-specific code (Java, in my example).
