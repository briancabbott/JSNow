Hi,

just for the archives: This worked with small one small modification
regarding the EOF-Token:

TEXT:
  {lxstate == State.NORMAL && input.LA(1) != curspecial}?=>.
{
 while (input.LA(1) != curspecial && input.LA(1) != CharStream.EOF) {
  input.consume();
 }
}
;

Thanks again
Frederic
