The second works--you can prove that by mathematical
induction.  That is
recursion level            result
0                          X
1                          XYX
2                          XYXYX
...
n                          X (YX)+n
n+1                        XYN[n] = X (YX)+n+1

where N[n] is N from recursion level n and (YX)+n is n
copies of YX.
