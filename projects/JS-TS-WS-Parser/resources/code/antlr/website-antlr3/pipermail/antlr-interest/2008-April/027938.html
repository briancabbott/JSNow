Try

commandline : command Blank argument ;  // allows for other commands  
than just 'command'
command : CMD ;
arguments : (command | Blank | Anything)* ;

CMD : 'command'
Anything : ~NewLine ;
NewLine : '\r' '\n'? | '\n' ;

That might be what you need.
