issues with what I consider to be a very simple SQL. Could someone

Exception in thread "main" line 10:2: expecting CLOSE_PAREN,

------------

FROM
    c.abcd

(
SELECT cr.no FROM c.cr cr
union
SELECT a FROM abc where a=b
)

    c.abcd

)
