Keep it as a rule of thumb to make the Lexer simple. This should work.

rule: stat EOF ;
stat: ID*;
fragment LETTER : ('a'..'z' | 'A'..'Z') ;
ID: LETTER ;
WS: (' ' | '\t' | '\n' | '\r' | '\f')+ {$channel = HIDDEN;} ;
