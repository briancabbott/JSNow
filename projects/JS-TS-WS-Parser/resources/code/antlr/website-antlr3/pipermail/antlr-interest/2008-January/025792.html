I think you have to explicitly pass the template up.  Try:

statement: foo | (help { $statement.st = $help.st} );

help: HELP -> help();

or maybe you can do:

statement: st=foo | st=help
                 { $statement.st=$st; };

help: HELP -> help();

I'm fairly certain I had the same problem in the past.
