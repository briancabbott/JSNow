>>3. {{{} }}} or

>>options { k=3; charVocabulary = '\u0020' .. '\u007e';}
>>CMT: "{{{"
>>        (
>>            {LA(3)!='}'}?   "}}"
>>        |   {LA(2)!='}'}?   '}'
>>        |                           ~'}'
>>        )*
>>        "}}}";
>>
>>... but as I'm new to antlr I'm not quite sure what I'm doing here.
