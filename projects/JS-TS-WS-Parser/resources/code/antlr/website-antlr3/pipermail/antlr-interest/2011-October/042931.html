Looks like you need:

group		:	(OPEN
				space*
				(
				sg=group space* |
				p=plaintext space* |
				((c1=command)+ space) |
				(e=escape) space*
				)+
				((c2=command)+)?
			CLOSE) -> ^(GROUP ^(GROUP $sg)* $p+ $c1+ $e+ $c2+)

At a rough guess.
