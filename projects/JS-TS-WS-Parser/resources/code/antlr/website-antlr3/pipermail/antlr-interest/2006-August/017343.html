Rightoh.  Well I just tried writing it out the 'long' way...

methodDefinition[AST mods]
	:	'function' (mode='get'|mode='set')? IDENT
		parameterDeclarationList
		typeExpression
		(block)?
		-> ^(METHOD_DEF {$mods}
		                mode? IDENT
				parameterDeclarationList
				typeExpression
				block?)
	;

...and this seems to pacify it.  I'll use that for the moment.
