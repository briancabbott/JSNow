> I'm also having problems with C target.. but I think you can solve this 
> with specifying ASTLabelType.
> 
> e.g.
> grammar Cee;
> 
> options{
>  
>     backtrack   = true;
> 
>     language    = C;
>     output    = AST;
> 
>     ASTLabelType= pANTLR3_BASE_TREE;   
> }
>
