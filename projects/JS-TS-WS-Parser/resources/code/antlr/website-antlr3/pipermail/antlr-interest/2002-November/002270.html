Yes, that works thank you.

One of the syntactic predicates can be removed.

class ExampleParser extends Parser;
options { k = 2;}

table
   : ( INTEGER
       ( (SPACE NEWLINE) => SPACE
       | SPACE INTEGER
       )*
       NEWLINE
     )+
   ;

[The original desired syntax was
  table: (INTEGER (SPACE INTEGER)* (SPACE)? NEWLINE)+;]
