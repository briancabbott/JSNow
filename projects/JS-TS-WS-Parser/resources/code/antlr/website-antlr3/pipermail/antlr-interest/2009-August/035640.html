Maybe you can use a setter to set what 'ifKeyword' is and use that in a
semantic predicate.

Something like:

@members {
	String ifKeyword;

	Public void setIfKeyword(String str) {
		this.ifKeyword = str;
	}
}

ifKeyword
	:	{ LT(1).getText().equals(ifKeyword) }?=> /*nothing*/
	;

Lexer will lex 'if' as just an ID.

Hope that helps.
