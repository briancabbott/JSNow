I think that the trick you are looking for is:

grammar harry;

start 
	:	(number_stuff)+
	;
	
number_stuff
	: (FLOAT | NUMBER) (DOT ID)?
	;
	
ID
	: ('a'..'z' | 'A'..'Z')+
	;
	
fragment
FLOAT
	: '0'..'9'+ '.' '0'..'9'
	;
	
NUMBER
	: ('0'..'9')+
		(
			 ('.' '0'..'9')=> ('.' ('0'..'9')+)
			 	{ $type = FLOAT; }
			|// Just an integer
		)
	;

DOT
	: '.'
	;
	
OTHER
	: . {$channel = HIDDEN; }
	;

If whitespace is significant to the parser then pass it back of course.
