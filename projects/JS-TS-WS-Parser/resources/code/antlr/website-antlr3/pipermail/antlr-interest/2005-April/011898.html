I think this grammar solves the ambiguity issue...

----------
class testParser extends Parser;

// want to find an unambiguous version of:
// suf : ( x | y )* ( ( x z ) | y ) ;

suf :
	( x ( suf | z ) )
    |   ( y ( suf )? )
;

x : X ;
y : Y ;
z : Z ;
----------
