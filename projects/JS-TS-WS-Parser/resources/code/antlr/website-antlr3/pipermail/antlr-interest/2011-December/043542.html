Hi James,

Try this:

STRING_LITERAL: '\'' (~'\'' | '\'\'')* '\'';

Note that the string above may contain line breaks. If you don't want that,
do:

STRING_LITERAL: '\'' (~('\'' | '\r' | '\n') | '\'\'')* '\'';

Regards,

Bart.
