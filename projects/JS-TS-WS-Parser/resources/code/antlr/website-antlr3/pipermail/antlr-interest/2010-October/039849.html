This is something stupid, I guess. I have a grammar like this below
and I would like to know why

"1+1-1" works

and

"1-1+1" does not work (NoViableAltException)

Leo K.

>>>>>>>>>>>>>>>>>

grammar Expr;

@header {
}

@members {
}

prog:   stat+ ;

stat:   comp NEWLINE
    |   NEWLINE
    ;

comp
    :   e=expr
        (   '>' expr
        |   '<' expr
        |   '=' expr
        )*
    ;

expr
    :   e=multExpr
        (   '+' multExpr
        |   '-' multExpr
        |   '*' multExpr
        |   '/' multExpr
        )*
    ;

multExpr
    	:	atom
    	(    	atom
    	)*
    ;

atom
    :   INT
    |   ID
    |  '(' comp ')'
    ;

ID  :   ('a'..'z'|'_')+ ;
INT :   '0'..'9'+ ;
NEWLINE:'\r'? '\n' ;
WS  :   (' '|'\t')+  ;
