Remember antlr.markmail.org as it will give you lots of examples of this. You need:

fragment DIGITS : ('0'..'9')+ ;
DATE
: DIGITS
  (
      (SLASH DIGITS SLASH DIGITS)=> SLASH DIGITS SLASH DIGITS
    | {$type = DIGITS; }
;

SLASH : '/' ;
