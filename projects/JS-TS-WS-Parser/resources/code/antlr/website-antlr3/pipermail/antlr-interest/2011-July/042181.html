Hi Folks,
I forgot to mention.  The compoundExpression rule has options{backtrack 
= true;}  set that is it reads like

compoundExpression
options {backtrack = true;}
:
      ^(sclarSubExpression  conversion_clause)
     | sclarSubExpression
;
