I see where I went wrong.  I needed to include the scope in the rule  eg:

ruleA returns [int result]
scope ParserGlobals;
@init  {
  $ParserGlobals::x = 0
}
  : ruleB { $result = $ParserGlobals::x; }
 ;

Thanks for the pointer.
