NEWLINE :
	(
		'\n' {if (!prevWasReturn) newline();}
		| '\r' { newline(); }
	);

then override match() in your lexer and do something like this:

match(?)
{
	super.match();
	if matched character was '\r' then prevWasReturn=true;
	else prevWasReturn=false;
}

Give it a try & let me know.
