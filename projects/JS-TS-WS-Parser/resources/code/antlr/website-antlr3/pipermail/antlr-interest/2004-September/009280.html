You can try this dash is nothing but the UNICODE so simply add 
charVocabulary='\u0000'..'\uFFFE'; in your options and declare range
in lexer like
IDENTIFIER :
	( '\u0080'..'\ufffe' | 'a'..'z' | 'A'..'Z');
