You do not have to list RESERVED as a rule, just use it as a string in 
the parser.

For example:

...
parser_rule_1
   : "keyword"  IDENTIFIER
   ;
...
class FooLexer extends Lexer;
options {
	testLiterals=false;	
}
...
IDENTIFIER
options{testLiterals=true;}:
...

You can use java.g as a reference.
