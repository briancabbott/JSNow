> Consider the following grammar:
> -----------------------------------------------------
> lexer grammar test;
>
> IF	 :	{getCharPositionInLine()==0}? => 'IF';

>
> NOT	 :	'NOT';	
>
> Identifier	:	('A'..'Z'|'a'..'z')+;
> WS	:	('\t'|' ')+;

> command. As I am not interested in all the commands, I decided to
