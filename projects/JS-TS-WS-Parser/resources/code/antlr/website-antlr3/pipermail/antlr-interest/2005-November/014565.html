I have found a working solution in case someone faces the same issue 
sometime later. Thanks for your replies!

AnyTextBlock
    :
    (
    '\0' .. '\u00FF'
    {
        if (LA(1) == '%' && LA(2) == ']'){
            break;
        }
        else if (LA(1) == '[' && LA(2) == '%'){
            break;
        }
    }
    )+
    ;

Cheers,
Dimitrios
