Hmm, I got it to work like:

primary_expression!
{ AST curAst = null;}
 :    ps:primary_start
      {curAst = ps_AST;}
     (
         ma:member_access
            {     curAst.setFirstChild(ma_AST);
                   curAst=ma_AST;
            }
     )*
    {#primary_expression = #ps;}
 ;

Maybe there is an easier way? Maybe I 'dont want' this in the first
place...?
