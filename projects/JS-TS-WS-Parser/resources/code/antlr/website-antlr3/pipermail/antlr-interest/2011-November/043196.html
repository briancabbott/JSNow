Hi,
I think I've found the reason . I've written a rule as follow
numeric_literal
    :    '0'..'9' ('0'..'9')*
    |    ('0'..'9')* '.' '0'..'9' ('0'..'9')*
    ;
which will be translated to MATCHRANGE in Parser , if I change it to this :
numeric_literal
    :    INT
    |    FLOAT
    ;
INT
    :    '0'..'9' ('0'..'9')*
    ;
FLOAT
    :    ('0'..'9')* '.' '0'..'9' ('0'..'9')*
    ;
will OK. Many thanks.
