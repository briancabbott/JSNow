Hi Terence,

Thanks for the reply. I managed to work around the problem by do this-

unary returns [exp::Exp* e]	
	: p=primary { $e = $p.e; }  (s=selector[e] { $e = $s.e;} )* 
        ;

I tried $p as well and it ANTLR doesnt seem to accept it.

Carter.
