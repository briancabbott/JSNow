>

> { #opt_pri = #([NICE, "nice"], #priVal); 

I have problems sometimes when I don't use "#" everywhere, so
you might try changing from this
	#( [NICE, "nice"], #priVal)
to this
	#( #[NICE, "nice"], #priVal)
