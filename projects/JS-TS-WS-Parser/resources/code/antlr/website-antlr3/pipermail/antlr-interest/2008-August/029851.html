I would think something like this would work

 adjective
        : 
 { String name = ""; } 
    (ADJECTIVE_PREFIX 
      { name += $prefix.text; } )? 
    (NAME
      {name += $NAME.text}) -> 
    {new CommonTree(new CommonToken(ADJECTIVE, name)))}
        ;
 
Basically, you build up name as you go.

I haven't tested this, but it seems like it should work and be clearer.
