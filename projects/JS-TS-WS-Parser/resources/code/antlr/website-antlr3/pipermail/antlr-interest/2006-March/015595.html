Hi Andrew,

Rules can return values:

rule1 returns [String text=null]
: 
  { text = LT(1).getText(); }
  ( "foo" | "bar" | "baz" ) 
;

rule2 
{
  String text1 = null;
  String text2 = null;
}
: x 
  text1=rule1  { if( text1.equals( "foo" ) ) System.out.println( "FOO" ); } 
  y 
  text2=rule1 { if( text1.equals( text2 ) ) System.out.println( "==" ); } 
;
