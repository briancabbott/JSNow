There is an ambiguity because of the newline() action. "\r\n" can be 
interpreted as either
  "\r\n" { newline(); } or
  '\r' { newline(); } '\n' { newline(); }

I think that either phrasing newline whitespace as
  ('\r')? '\n { newline(); }
  | '\r' { newline(); }

or setting the greedy option to "true" (preferred) works.
