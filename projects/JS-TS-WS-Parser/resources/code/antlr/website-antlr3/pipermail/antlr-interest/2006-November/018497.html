I gather the following rule is not valid, but it produces no complaint 
from ANTLR v3.0b5.

DistinctObject
returns [ String do ]
  : '"' ( ~( '"' | '\\' ) | '\\' '"' | '\\' '\\' )* '"'
    {
      $do = getText().substring(1, getText().length() - 1);
    }
  ;
