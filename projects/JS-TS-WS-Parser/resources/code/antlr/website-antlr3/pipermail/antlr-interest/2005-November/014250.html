Here's what I use for SL comments -- it's for SQL but 
should work for c++.

I'm generating c++ code on LINUX and I built
antlr-2.7.5 
from scratch.

// Single-line comments
SL_COMMENT
	: ('-' '-'	(~('\n'|'\r'))* ('\n'|'\r'('\n')?) ) => 
		'-' '-'
		(~('\n'|'\r'))* ('\n'|'\r'('\n')?)
		{ newline(); _ttype =
ANTLR_USE_NAMESPACE(antlr)Token::SKIP; }
     | '-' '-'(~('\n'|'\r'))*
       {_ttype =
ANTLR_USE_NAMESPACE(antlr)Token::SKIP; } 
	;
