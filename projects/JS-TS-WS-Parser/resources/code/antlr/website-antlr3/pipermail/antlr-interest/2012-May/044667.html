I like them both. You may find that you need to have a few permutations 
of the logo handy. I think that the rectangular 
(http://antlr.org/download/antlr-black2.pdf) one would look best on the 
header of the website, but in other places you will need to have a much 
more square logo. For example, if ANTLR were to have its own Facebook or 
Google+ page, they require a square image for the logo. The same goes 
for an application icon, or a favicon for the website. You will have to 
see in these cases if it looks better to just use the red circle by 
itself, or with ANTLR written underneath.
