>>My solution:
>>
>>REAL_OR_INT
>>    : ( INT '.' ) => REAL { $setType(REAL); }
>>    | ('-' INT '.' ) => REAL { $setType(REAL); }
>>    | INT { $setType(INT); }
>>    ;
>>
>>protected INT options { paraphrase = "a constant integer"; }
>>    : ('0' .. '9')+
>>    ;
>>
>>protected REAL options { paraphrase = "a floating point value"; }
>>    : ('-')? INT '.' INT
>>    ;
>>
>>Hope it helps,
