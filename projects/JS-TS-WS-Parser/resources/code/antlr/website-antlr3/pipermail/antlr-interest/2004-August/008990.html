Yes,

I added:

protected DOT_DOT:  "..";
to the lexer and

interval:
   variable DOT_DOT variable
to the parser
This works fine for variables.
I ended up with:

(   integer DOT DOT integer ) => integer DOT DOT
integer

for the integer ranges.
