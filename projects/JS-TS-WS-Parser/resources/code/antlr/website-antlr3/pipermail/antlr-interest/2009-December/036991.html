>STR
>     :   '`' ( ESC_SEQ | ~('\\'|'"') )* '\'';

That double-quote should be a single quote.

>ESC_SEQ
>     :   '\\' ('b'|'t'|'n'|'f'|'r'|'"'|'\''|'\\')
>     |   UNICODE_ESC
>     |   OCTAL_ESC
>     ;
>
>OCTAL_ESC
>     :   '\\' ('0'..'3') ('0'..'7') ('0'..'7')
>     |   '\\' ('0'..'7') ('0'..'7')
>     |   '\\' ('0'..'7')
>     ;
>
>UNICODE_ESC
>     :   '\\' 'u' HEX_DIGIT HEX_DIGIT HEX_DIGIT HEX_DIGIT
>   ;
>
>HEX_DIGIT : ('0'..'9'|'a'..'f'|'A'..'F') ;

All of these should be marked as fragments.
