If I have the following in a lexer:

	FOO
		: { isFoo() }? "foo"
		;

	BAR
		: ('a'..'z')+
		;

What I want is that if isFoo() is false, then the more generic rule BAR to be used.  Is the above sufficient or must I do something like:

	FOO
		: { isFoo() }? "foo"
		| BAR { $setType( BAR ); }
		;

	BAR
		: ('a'..'z')+
		;

that?
