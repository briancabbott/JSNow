As long as we're on that subject, might I point out Trac (http://www.edgewall.com/trac/). It is a combined wiki, a bug tracker and source control frontend. The major benefit is easy crosslinking between bug reports/issues, source code and the wiki.

And, the source is open and quite simple (writtend in Python) which makes modifications easy. If Confluence is anything like Atlassian Jira, extensions and source modifications require a lot of work.
