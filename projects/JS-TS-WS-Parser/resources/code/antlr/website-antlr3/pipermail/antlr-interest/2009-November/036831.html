How are you using these lexer rules? Check the parse tree in  
ANTLRWorks interpreter or debugger. You should be able to determine  
what lexer rule is being used for your input. This works for me with  
input 1.0


prog	:	start* ;

start:	NUM | EXP | ENUM ;

fragment DIGIT : '0'..'9' ;
fragment INT   : DIGIT + ;
NUM     : INT ( '.' INT)? ;
EXP     : ( 'E' | 'e' ) ( '-' | '+')? NUM  ;
ENUM    : NUM ( EXP )? ;


steve
