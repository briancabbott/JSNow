Hi Søren, have you considered the below rule. It may help to have a cleaner stmt rule:

WS  :   ( ' ' | '\t' | '\u000C' | '\r' | '\n')  { $channel=HIDDEN; }
    ;

Then the rule will look like:

stmt    :       variable_decl ';' | method_call ';' | assignment ';'
        ;
