Oh, I see the problem.  It's that you have SEMI in your expr rule.  When
atom tries LPAREN expr RPAREN notice that expr requires a SEMI (before
LPAREN).
Try:

atom      : INT 
          | LPAREN^ sumExpr RPAREN! ;

Monty
