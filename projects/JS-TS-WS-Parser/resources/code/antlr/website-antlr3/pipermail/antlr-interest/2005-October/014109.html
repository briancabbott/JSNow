last token consumed:

    int currentLine = 0;

        {
            currentLine = LA(1);
            base.consume();
        }

I have then defined semi:

    semi
        :    {currentLine == LT(1).getLine()}? SEMI
        |    (SEMI)?
        ;

This seems to work, but gives me lots of disambiguation warnings. What 
do you think?
