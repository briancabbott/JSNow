Try:

  {(input.LA(1) == '\\' && ( input.LA(2) == ' ' || input.LA(2) ==
'\t'))}? '\\'
| etc.

The lexer is probably thinking that it does not need the predicate as
you have expressed it.

Epsilon means an empty alt:

fred	: X
	| // Epsilon
	;
