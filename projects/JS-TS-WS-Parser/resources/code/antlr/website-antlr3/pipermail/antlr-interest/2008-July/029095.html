Thanks Johannes and Gavin. It works and it seems to me that it can even be
simplified to this:

rule : (id -> id) ( '+' id -> ^('+' $rule id))*;

Correct me if I'm wrong. The simpler the rewrite the easier it is to
reconstruct...
