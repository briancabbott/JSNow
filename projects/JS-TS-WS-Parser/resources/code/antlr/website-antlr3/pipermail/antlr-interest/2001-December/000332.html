Can I use 'break' statement inside the loop like this:

class myParser extends Parser;
somerule
{ bool bQuit=false; } 
      :  SOME_TOKEN 
            ( 
              bQuit = otherRule
              { if (bQuit) break; }
            )+
           END_TOKEN
        ;
otherRule returns [bool bRet]
{ bRet = false; } 
                             : A B
                               ( C | D {bRet=true;}
                               );

out of a loop after I have seen some special character?
