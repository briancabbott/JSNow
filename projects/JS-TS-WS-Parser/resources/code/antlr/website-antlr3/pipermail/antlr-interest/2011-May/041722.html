Hi Vijay,

You could grab all matched text in the `@after` block using the `getText()`
method:

COMMENT
@init{
  boolean isJavaDoc = false;
  System.out.println("Entering comment");
}
@after {
  System.out.println("Leaving comment, matched: " + getText());
}
  :  '/*'
     {
       if((char)input.LA(1) == '*') {
         isJavaDoc = true;
       }
     }
     (options {greedy=false;} : . )*
     '*/'
  ;


Regards,

Bart Kiers.
