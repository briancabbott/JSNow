try to combine the last two matches together

FloatingPointLiteral
   : (Sign)?  ('0'..'9')+ '.' ('0'..'9')* Exponent? FloatTypeSuffix?
   | (Sign)?  '.' ('0'..'9')+ Exponent? FloatTypeSuffix?
   | (Sign)?  ('0'..'9')+ Exponent? FloatTypeSuffix?
   ;

it works for my grammar, where Sign rule is just:

Sign : ('+'|'-') ;
