You started from an empty workspace ? Gah I now see that I still had the
old MSVC projects in contrib, could have sworn I had deleted those.

Start with an empty project Win32 static library, Enable Run-Time Type Info
add all the CPP files from the lib/cpp/src directory except the dll.cpp.
Add lib/cpp to your include path. (might need full path for the latter)
After that things should compile. Also for the ADA parser add the lib/cpp
directory to the searchpath for include files.
