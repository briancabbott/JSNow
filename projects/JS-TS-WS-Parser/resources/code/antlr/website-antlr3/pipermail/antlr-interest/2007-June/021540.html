As you haven't sent your grammar with your question I have to improvise
the snippets. Create first an imaginary LABEL token and have rules like
this:

label
   :   ID[false] ':' ID[true]
   ;

ID[boolean isID]
   :   ('a'..'z'|'A'..'Z')+ { if (!isID) $type = LABEL ; }
   ;

Hope this helps!
