Dear Mark,

You can add the additional newline() function into your lexer rule, that 
fixed it for me. Below is my grammar for white space.

//========================================================[WHITE SPACE]
WS   
    :    (SPACE
    |    '\t'
    |    '\n'
    |    '\r')
        {
            $setType(Token.SKIP);
            newline();
        }
    ;

Hope This Helps
Vladimir Sutskever
