Don't try to construct syntactical context in the lexer basically:

grammar T;
 
options {
 	language=CSharp;
 	backtracking=true;
}
 
ID : ('a'..'z')+;
 
LPAREN : '(';
 
SELECT : 'select';
 
prog: (ID | LPAREN SELECT?)+ ;
