> Except the "range" rule needs to be a lexer rule as well.  So
> "RANGE" instead.  And if you're defining DOT like that anyway, you
> might as well write it like this:
> 
> fragment DOT : '.';
> 
> RANGE
>    : DOT
>      ( (DOT) => DOT
>      | {$type = DOT; }
>      )
>    ;
