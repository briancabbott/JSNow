Hi Patrick,

The range operator, `..`, only works in lexer rules, not in parser rules as
you're doing. Capitalize the `letter` rule to make it a lexer rule:

grammar failuretest;
start:          letter+;
Letter:         'A' .. 'B'; // or did you mean 'A'..'Z'?

and then try again.

Regards,

Bart.
