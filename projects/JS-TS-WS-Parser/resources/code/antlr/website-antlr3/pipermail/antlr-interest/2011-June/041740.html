Hi Nilo,

The grammar:

grammar Brasil;

parse
  :  WORD EOF
  ;

 WORD
   :  ('\u00c0'..'\u00ff' | 'a'..'z' | 'A'..'Z' | '-')+
   ;


parses the input "não" just fine in ANTLRWorks.
