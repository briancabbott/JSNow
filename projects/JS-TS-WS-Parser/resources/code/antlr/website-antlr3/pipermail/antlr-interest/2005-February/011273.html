Unfortunately, not.  You can do something like

rule
{ AST ast }
     :
     (    g1:GT { ast = #g1; }
     |    g2:GE { ast = #g2; }
     |   ...
     ) expr { anAction(op.getText()); }
     ;

but you cannot label subrules or even use the same label on alternatives.
