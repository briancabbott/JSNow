On Nov 3, 2004, at 4:57 AM, thoth2487 wrote:

> 	
> // identifier
> IDENT
>   	: 	('a'..'z' | 'A'..'Z')
>   		('a'..'z' | 'A'..'Z' | '0'..'9')*
> 	;
> 	
> // label
> LABEL
>   	: 	('a'..'z' | 'A'..'Z')
>   		('a'..'z' | 'A'..'Z' | '0'..'9')*
>   		':'
> 	;

Factor it into only one rule:
	
// identifier or
// label
IDENT
   	: 	('a'..'z' | 'A'..'Z')
   		('a'..'z' | 'A'..'Z' | '0'..'9')*
   		':' {$setType(LABEL);}
	;

be sure to add LABEL to your tokens{} section
