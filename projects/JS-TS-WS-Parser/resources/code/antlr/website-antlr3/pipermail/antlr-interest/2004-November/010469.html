Off the head:

You'd need to tell ANTLR that a null is expected.

define a lexer rule:

NULL_CHAR: '\u0000';

then define your parser rule like:

text:
  subrules ... ... ... NULL_CHAR! EOF! ;

This ought to solve the problem.
