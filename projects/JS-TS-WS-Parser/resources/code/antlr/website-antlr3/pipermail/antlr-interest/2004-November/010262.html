Somebody has the main and .g file looking like one.  Make this a 
separate file:

import antlr.*;
public class Main {
  public static void main(String[] args) throws Exception {
    ExprLexer lexer = new ExprLexer(System.in);
    ExprParser parser = new ExprParser(lexer);
    parser.expr();
  }
}
