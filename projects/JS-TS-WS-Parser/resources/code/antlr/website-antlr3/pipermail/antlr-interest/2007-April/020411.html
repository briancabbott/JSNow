Another knotty syntax example,

  ----8<----
  a=1
  [<<HERE, <<'THERE', <<-WHERE].each {|str| puts "value: #{str.inspect}"}
  foo #{a}
  HERE
  bar #{a}
  THERE
  foo
    and
      bar
        WHERE
  ---->8----

When run,

  value: "foo 1\n"
  value: "bar \#{a}\n"
  value: "foo\n  and\n    bar\n"
