I am not an expert in Antlr's meta-language, but it seems that the
returns keyword must go outside the []'s.

So I think do this:

file returns [string s] {s = "something" ; }: ( ( (design_unit )+ ) ? )
 EOF ;

instead of this:

file [returns string s] {s = "something" ; }: ( ( (design_unit )+ ) ? )
 EOF ;

Hope this helps....
