How about:

fragment
TAG_GUTS: LETTER+
		{ emit(); }
;
HTML_TAG: '<' TAG_GUTS '>'
;

You may need to set the token type for the emitted token.

You can also do: xxx=FRAGRULE  and set the token text to $xxx.text
