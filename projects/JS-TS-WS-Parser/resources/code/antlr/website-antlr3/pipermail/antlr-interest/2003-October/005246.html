>
> To fix the java.tree.g grammar, just add typeDefinition to the  
> interfaceBlock production so it looks like this:
>
> interfaceBlock
>     :    #(    OBJBLOCK
>             (    methodDecl
>             |    variableDef
>             |    typeDefinition
>             )*
>         )
>     ;
>
> Hope that helps.
