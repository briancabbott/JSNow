Hi James,

Something like this would do the trick:

lines
  :  (LineStartingWithPlus | LineNotStartingWithPlus | LineBreak)* EOF
  ;

LineStartingWithPlus
  :  '+' ~('\r' | '\n')*
  ;

LineNotStartingWithPlus
  :  ~'+' ~('\r' | '\n')*
  ;

LineBreak
  :  ('\r'? '\n' | '\r')
  ;

Bart.
