Olivier:

In addition to the other answers you received, I think that creating another
rule is often the simplest way to do this, especially when rules begin to
get complicated.

  foobar
     : t:foobar2
  { doSomethingWith( t ); };

  foobar2
     : "function" | "subroutine" | "program"
     ;
