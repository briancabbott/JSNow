Hi Jason,

The fragment rule NONQUOTE hides the ID rule.
# is 35, ~ is 126
'a'..'z' and 'A'..'Z' is within 35..126

>  fragment
>  NONQUOTE
>    :' '..'!'|'#'..'~'|'\r'|'\n';

-Shaoting
