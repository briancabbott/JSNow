This works perfectly.  Thanks!  I don't know why I thought that syntactic predicates didn't work on lexer rules.


COMMENT : '#'
                         (
                              ( 'ABCD')=> 'ABCD'
                            | ~('\r' | '\n')* NEWLINE
                         )
                         { $channel = HIDDEN; }
                      ;



NEWLINE     : '\r' ('\n')? | '\n';
