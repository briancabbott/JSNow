Try this:
   #(IF_TOKEN t=expression
      ( {t == true}?
         val=expression .
      |
         // expression was false
         . val=expression
      )
   )

This uses the value of the expression, captured in t, to direct the parse to
only evaluate the leg of interest.

Hope this helps,
- Bryan
