Hi.  Damn yahoogroups sucks.  Post from home didn't make it.  Anyway, 
try this:

class L extends Lexer;

options {
	filter=true;
}

CMT : "--" ( PRAGMA | (options{greedy=false;}:.)* '\n' ) ;

protected
PRAGMA : "<pragma"> ... ;
