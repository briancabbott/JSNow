My language consists only of Letters (a..z) and a few tokens ("and", "or"). I am trying to match an expression "A and B" using the following
 
Parser
---------
myOp:
            atom (AND atom)*
       ;
 
atom: Letter
        ;
 
Lexer
-------
 
tokens { AND = "and";}
Letter: 'a'..'z'

myOp:
            (atom AND) => atom (AND atom)*
