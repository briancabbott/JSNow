The basic methodology is "refactor until the transformation is obvious".  For this one, the sequence is:

a :  b  ( '+' x = b  -> ^('+' $a $x) )* ;

goes to

a :   b  ( '+' x = b  -> ^('+' $a $x) )+ 
        |  b
    ;

after which you factor out

c : b  ( '+' x = b  -> ^('+' $a $x) )+ ;

which becomes

c :  c1 = a  '+' x = b -> ^('+' $c1 $x) ;

and substituting back into a gives

a :  ^( '+' a b )
        | b
    ;
