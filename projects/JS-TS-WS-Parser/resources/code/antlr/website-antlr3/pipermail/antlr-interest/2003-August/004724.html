I'd anyway recommend this other html code for making the link:

<a href="http://www.antlr.org">
<img src="http://www.antlr.org/images/antlr-link.gif"
 border=0 alt="ANTLR logo"
 width=88 heigth=31></a>

This solves some problems, like borders, etc. Note there can not be
white spaces between the <img ...> and the closing </a>: spaces appear
underlined on links (so there's a 1 space underlined after the logo).

antlraux.sf.net is already showing the logo!
