I have a situation where the following string may arise

blah CLIPPED 

I want to represent this in an AST as:

LS_CLIPPED
   |
   blah

factor
  : v:variable (l:leftSidedOperators[#v] { #factor=#l; })?
  ;

leftSidedOperators[AST operator]
