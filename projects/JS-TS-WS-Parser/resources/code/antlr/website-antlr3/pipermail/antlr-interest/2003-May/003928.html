Did you set the astFactory ?

E.g. as in the exprAST example in examples/cpp/exprAST:

ExprLexer lexer(cin);
ExprParser parser(lexer);

parser.initializeASTFactory(my_factory);
parser.setASTFactory(&my_factory);

parser.expr();
