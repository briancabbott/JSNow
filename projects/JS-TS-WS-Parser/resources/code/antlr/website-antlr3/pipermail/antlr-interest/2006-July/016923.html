Small bug inside your build.xml file, the generated jar for antlr
doesn't provide the file "ANTLRCore.sti" :

  <fileset dir="${src.dir}" includes="**/*.stg, **/*.st"/>
should become
  <fileset dir="${src.dir}" includes="**/*.stg, **/*.st, **/*.sti"/>
