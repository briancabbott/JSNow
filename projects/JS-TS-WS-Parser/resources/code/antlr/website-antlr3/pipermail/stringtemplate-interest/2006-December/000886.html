You could try something like this...

ArrayList list = new ArrayList();
list.add("10");
list.add("20");
list.add("30");
StringTemplate template = new StringTemplate("$data:{$i$ == $it$\n}$");
template.setAttribute("data", list.iterator());
System.out.println(template.toString());

Outputs...

1 == 10
2 == 20
3 == 30
