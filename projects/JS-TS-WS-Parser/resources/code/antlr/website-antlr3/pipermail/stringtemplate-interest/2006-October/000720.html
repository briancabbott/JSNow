> Any pointers would be welcome! Or an example with strip() in it... c

Typed-in-mail-disclaimer etc.:

select(mappings, param) ::= <<
SELECT
	<strip(mappings):{map | <strip(map.membersWithoutKeys):{mem |  
<map.table>.<mem.column>}>}; separator=",\r\n">
...etc.
 >>

should do the trick.
