whitespace is ignored *inside* expressions so that works now :)

What about your

<table border=1>
$users:row()$
</table>
 >>

foo(users) ::= <<<table border=1>
$users:row()$
</table>
 >>

which is ugly.

Maybe then with some templates we can make a consistent by ignoring  
one space or newline:

{x |
blort
}
