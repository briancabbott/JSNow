Don't use $$ when inside an if, because you are already inside $$. Eg...
$if(it.countIsOne)$
...and NOT...
$if($it.countIsOne$)$

Also, templates can have multiple parameters. Eg...
variable(type,name) ::= <<
   <b>$type$</b> $name$ = $typeInitMap.(type)$;$\n$
 >>
Take from this page...
http://www.antlr.org/wiki/display/ST/simple_group_example
