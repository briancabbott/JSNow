Hi,

>>> st = StringTemplate("$x.(y)$")
>>> st["x"] = { "0": "zero", "1": "one" }
>>> st["y"] = 1
>>> print unicode(st)
one
>>> st = StringTemplate("$x.(y)$")
>>> st["x"] = { "0": "zero", "1": "one" }
>>> st["y"] = 0
>>> print unicode(st)

>>>

Expected: “zero”.

Maybe it’s an “if something:” in a place where “if something is not None:” 
should have been used. The documentation is explicit in treating only "" and 
None as null values.
