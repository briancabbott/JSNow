Menu Help

  • Help: access the help of ANTLRWorks
  • Submit Statistics: submit the usage statistics of ANTLR and ANTLRWorks to the ANTLR server. These statistics are used to improve both ANTLR and ANTLRWorks. Nothing personal is sent and all the data can be viewed before sending them.
  • Send Feedback: open the feedback web page of ANTLRWorks.
  • Check For Updates: check if a new version of ANTLRWorks is available. Note that ANTLRWorks can perform this check automatically (see Preferences).
