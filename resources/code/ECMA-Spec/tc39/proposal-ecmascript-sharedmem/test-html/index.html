These test cases run as you load them and test that sharing restrictions are implemented correctly:

  • test_sharing_window_and_worker: Sharing memory between Window and Worker works (both directions)
  • test_sharing_worker_and_nested_worker: Sharing memory between Worker and its nested Worker works (both directions)
  • test_sharing_sharedworker_and_nested_worker: Sharing memory between SharedWorker and its nested Worker works (both directions)
  • test_sharing_window_and_sharedworker: Sharing between Window and SharedWorker does NOT work (either direction)
