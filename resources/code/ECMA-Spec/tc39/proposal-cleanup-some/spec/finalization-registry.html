FinalizationRegistry.prototype.cleanupSome ( [ _callback_ ] )

The following steps are taken:

1. Let _finalizationRegistry_ be the *this* value. 1. Perform ? RequireInternalSlot(_finalizationRegistry_, [[Cells]]). 1. If _callback_ is present and IsCallable(_callback_) is *false*, throw a *TypeError* exception. 1. Perform ? CleanupFinalizationRegistry(_finalizationRegistry_, _callback_). 1. Return *undefined*. This section is to be treated identically to the "Annex B" of ECMA-262, but to be written in-line with the main specification. See HTML issue #5446 for information about cleanupSome on browsers.
