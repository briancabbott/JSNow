When the `deleteAll` method is called, the following steps are taken:

1. Let _len_ be the actual number of arguments passed to this function. 2. Let _items_ be the List of arguments passed to this function. 3. Let _set_ be the *this* value. 1. If Type(_set_) is not Object, throw a *TypeError* exception. 4. Let _k_ be 0. 5. Let _remover_ be ? Get(_set_, "delete") 6. If IsCallable(_remover_) is *false*, throw a *TypeError* exception. 7. Let _allDeleted_ be *true*. 7. Repeat while _k_ < _len_ 1. Let _value_ be _items_[_k_]. 2. Let _wasDeleted_ be Call(_remover_, _set_, _value_) 3. Set `allDeleted` to be `allDeleted` && `wasDeleted` 4. Increase _k_ by 1. 8. Return ToBoolean(`allDeleted`);
